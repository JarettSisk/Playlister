# Playlister

Playlister is a web app that allows users to create and share music playlists for any event.

> Live demo [_here_](https://jarett-sisk-playlister-app.herokuapp.com/).

## Technologies used

Back-end:
- Python
- Flask
- Jinja
- WTForms

Database:
- Postgresql
- SQLAlchemy

Front-end:
- HTML
- CSS
- JavaScript
- Axios

API:
- [Itunes API](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html)
